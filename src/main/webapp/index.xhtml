Cargo Tracker

Welcome to the Cargo Tracker application.

There are two web interfaces available:

  • Public cargo tracking.
  • Administration of booking and routing.

The Incident Logging application, that is used to register handling events, is a separate standalone interface.

Please visit the project website for more information about the application and it's goals.
