Cargo Tracker

Welcome to the Cargo Tracker application.

There are three web interfaces available:

  • A public landing page for tracking cargo.
  • An administrative landing page for booking, routing and managing cargo.
  • The Incident Logging application that is used to register cargo handling events at ports (this interface is optimized for mobile devices).

Please visit the project website for more information about the application and it's goals.

This project is an effort by a few Java EE and DDD practitioners within Oracle Corporation
